Question 3
Given the following transfer function, calculate the step response, impulse response and the time response with a square wave input of period 8s with amplitude 1;

`G(s)=\frac(16)(s^2+3s+16)`

Hint: the syntax of generating square wave in Matlab is 
                        f = square(T);
For example, to generate a square wave of period 1s with amplitude 1, we can use the following scripts. 
                        t = 0:0.001:10;
                        f = square(2*pi*t);
                        plot(t,f);
Note that if you are using Octave, you need to add "pkg load signal;" at the beginning of the script. Run the scripts, we can get the figure of the square wave;
Question 4
Consider the following mass-damper-spring system, we can depict it with an ODE `m\ddot(x)+c\dot(x)+kx=f`.
Suppose we have `m=1,c=2,k=1`, create a simulation to show the response of `x` under a constant force `f=1`.

Hint: For a given second-order system `\ddot(x)+b\dot(x)+c=u`, denote `x_1=x, x_2=\dot(x)` then the system can be rewritten as

`\dot x_1=x_2`
`\dot x_2=-bx_1-c+u`
